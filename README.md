## üîê What is Quercus?

Quercus is an cryptocurrency with a total supply of **52,500,000**. Built on Bitcoin's proven codebase and secured by SHA-256 mining, featuring 5 minute avg block times.  Quercus represents the ultimate experiment in digital scarcity and immutability.

**The Quercus Concept:**
- **1,500,000 premine at block 1** for initial distribution and development
- Repository will be **permanently archived** after block 1
- No future code changes or governance
- Pure, immutable, fixed forever

## üéØ Why Quercus?

Quercus represents a radical experiment in cryptocurrency design:

- ‚úÖ **No Governance Drama** - No debates, no forks, no community splits
- ‚úÖ **No Protocol Upgrades** - What you see is what you get, forever
- ‚úÖ **No Inflation Surprises** - 52,500,000. That's it. Forever.
- ‚úÖ **Fair Launch with Premine** - 1,575,000 QCX premine (3% of supply) for ecosystem development

## ‚ùì Frequently Asked Questions

### What is the premine for?
The 1,575,000 QCX premine (3% of total supply) at block 1 provides initial liquidity for exchanges, funds community development, and ensures ecosystem sustainability. This is a transparent, one-time allocation locked in the blockchain forever.

### Can the code be updated later?
No. The repository will be archived after block 1. No updates, no bug fixes, no changes. Ever. This is by design.

### Is this a joke/meme coin?
No. Quercus is a serious experiment in protocol immutability and scarcity while utilizing a 5 min avg block time. It's built on Bitcoin's proven codebase with SHA-256 security.

### How can I contribute?
Build tools, wallets, services, and applications. The protocol is fixed, but the ecosystem is yours to create. Everything must be built by the community.

### What makes Quercus different from Bitcoin?
- **Supply:** 52,500,000 vs 21,000,000  (Double Bitcoin)
- **Premine:** 1,575,000 for ecosystem development
- **Block Rewards:** 25 starting at block 2, while halving 2x as quickly as Bitcoin
- **Absolute immutability:** No updates after archive
- **Faster block times:** 5 min avg vs 10 min for Bitcoin
- **Community-driven:** No foundation or central authority

### Is it safe to use?
Quercus uses Bitcoin's proven codebase and SHA-256 algorithm. However, as with all cryptocurrencies, use at your own risk. See the [Disclaimer](#Ô∏è-disclaimer) section.
